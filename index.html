Google Doodle

Inspired by Google's JavaScript-powered Doodle marking Eadweard J Muybridge's 182nd birthday, this is a pure CSS alternative using transitions against a background-image sprite. As a result, the CSS3 animation behaves almost exactly as the JavaScript original.
